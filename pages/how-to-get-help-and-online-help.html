How to get HELP and online HELP

 

The #BIKES4ERP web application has a “Help” Icon in the top right corner of the screen, next to the “Logout” button.  A click of the icon will take you to the “Help” guide. Should a system user get confused at any point, this “Help Guide” gives the user further explanation on system procedures.

The “Help” icon will take you directly to the point in the guide that corresponds to the current screen you are on; this will serve as online help.
