Create Business Rule Screen

 

Screen Description

This is the CreateÂ  Business Rule screen of the #Bikes4ERP web application, accessed by the clicking “Plus” icon on the “ Business Rule Management” screen. It allows a Super Administrator to create a new Business Rule. The user will enter the required details and proceed to click the “Create” button.

If the user clicks on the Help option, Redirects Super Administrator to the Help document

5

Logout Button

If the User clicks on the Logout option, then it will log the Super Administrator out of the system and Redirects to the Home page
