QR Scan Screen

Screen Description

This is the QR Scan screen of the #Bikes4ERP web application, accessed by clicking “Report Fault” menu item on the Student menu icon. It allows a user to scan their Bicycle QR code to either check in a bicycle or report a fault for the bicycle.

 

 

 

Number

Item

Description

1

Navigation Bar: Home Button

Redirects the User to the Bikes4ERP Home page.

2

QR Scan Screen

Allows the user to scan their QR code by placing the code in front of the camera, If the code has a job due it will display a Checked-in dialog, if there is no Job it will redirect the user to the QR scan page

3

Help Button

If the user clicks on the Help option, Redirects User Administrator to the Help document

4

Logout Button

If the User clicks on the Logout option, then it will log the User out of the system and Redirects to the Home page
