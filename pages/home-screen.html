Home Screen

 

Screen Description

This is the Home Screen of the #Bikes4ERP web application, available on loading the web application. It is through this page the user will be able to login by clicking the “Log in” button in the top right corner. A student will also be able to report a fault by clicking on the “Student” menu icon and then selecting the “Report Fault” menu item.
