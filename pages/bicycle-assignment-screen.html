Bicycle Assignment screen

Screen Description

This is the Bicycle Assignment screen of the #Bikes4ERP web application, accessed by the clicking “Assign Bicycles” menu item on the “School Administrator” menu icon. It allows a School Administrator to search all the bicycles currently on the system, the user will also be able assign and unassign bicycles for students in their school.
